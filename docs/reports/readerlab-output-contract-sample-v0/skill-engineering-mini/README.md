# Skill/工程材料最小样章

这是按 ReaderLab 输出契约生成的 Skill/工程材料 reader-facing 最小样章，用来给后续纯读者审查“正文优先”的阅读体验。

它不是 product ready，不代表完整生成器能力，也不写入 LifeAtlas。当前只覆盖三个小模块：任务路由、证据边界、输出评估。覆盖目的不是证明完整包理解，而是验证 Skill/工程材料能否先呈现净化正文，再提供高阶讲解、设计资产卡和独立审计页。

## 文件

- `reader/01_任务路由与证据边界.md`：读者主样章。先放净化正文，再放成段高阶讲解、设计资产卡、必要提醒和可批注问题。
- `audit/source-notes.md`：审计页。集中放 source refs、location refs、machine/human 状态、降级/拒绝说明和自检结果。

## 边界

- 这是 Skill/工程材料，不是书籍样章。
- 主体是净化正文，不是摘要。
- 安装命令、路径、hash、机器状态和调试信息不进入读者主体。
- 人工阅读质量尚未验收。
