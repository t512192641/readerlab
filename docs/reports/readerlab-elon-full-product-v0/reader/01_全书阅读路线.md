# 全书阅读路线

## 正式标题映射

| 阅读单元 | 正式标题或可追溯标题 | 读者任务 |
|---|---|---|
| 前置材料 | 本书说明 / 前言 / 埃里克欢迎辞 | 确认材料性质：这是马斯克言论选编，引用和升格前要保留语境边界。 |
| 第一部分 | 第一部分：追寻目标 | 建立目标、价值、第一性原理和工程价值的入口。 |
| 第二部分 | 第二部分：极限硬核工作；旧 demo 映射名：部件 II：极致艰苦工作 | 把高目标转成责任、团队、沟通、算法、速度和制造系统。 |
| 第三部分 | 第三部分：企业建设 | 用 Zip2、PayPal、Tesla、SpaceX 检验前面的机制。 |
| 第四部分 | 第四部分：代表人类 | 把企业行动放到富足、AI、生存风险和多行星尺度下判断。 |
| 附录 | 69项核心马斯克法则 / 年表 / 推荐书单等 | 复盘和查阅，不抢占第一轮主线。 |

## 第一次阅读路线

1. 先读前置材料：确认这是选编文本，不把它误当连续论证。
2. 读第一部分：问“什么值得做”和“为什么第一性原理能改变路径”。
3. 深读第二部分：把“硬核工作”拆成责任、沟通、算法、速度、制造。
4. 读第三部分：看公司案例如何支撑或挑战这些方法。
5. 读第四部分：判断文明尺度是否让行动更清楚，还是让边界更容易被忽略。
6. 最后用附录复盘原则和时间线。

## 全书路线假设和已覆盖关系

这套路线已经覆盖整本书的结构和关键阅读单元；但它仍不是逐章精选译文。第一次阅读时，先用全书地图把主线立起来，再把第二部分的深读卡作为重点批注区。
