# ReaderLab Progress

## 口径

本文件是 ReaderLab 的工程进度板。它不追求精确数学，而追求每轮大迭代后能直观看到：

- 总体离最终目标还有多远。
- 本轮新增了什么能力。
- 本轮推翻或扣减了什么旧判断。
- 哪些模块可以 close，哪些仍在 working，哪些 blocked 或 not_started。

最终目标：ReaderLab 能稳定把复杂材料转成中文可读、可批注、可讨论、可学习、可沉淀的 LifeAtlas 阅读包，并且能证明输出质量不是空泛总结。

## Close / Working / Blocker

### Close

- 项目根目标和边界。
- 机器完成不等于人工验收。
- 主阅读页和审计附录分离。
- 不把仿型功能菜单化拼装。
- 每轮大迭代必须更新进度板。

### Working

- 结构化提炼：最小手工闭环和同题多路线对照已完成，下一步是 Combo-B 轻量模板、blind A/B 和第二类材料迁移测试。
- 片段深读与误读防护。
- 能力化判断。
- source/location 精度和来源复核。
- 完整阅读包的产品组织。

### Blocker / Not Started

- 正式图书/长文导入生成器。
- provider-backed 或 blind A/B output eval。
- 人工验收闭环。
- ReaderLab 正式 Skill 包装。
- LifeAtlas `300/600/800` 正式沉淀流程。
